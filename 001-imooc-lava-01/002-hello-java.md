# hello java

## source:
1. create `HelloWorld.java`
2. paste source code:
```java
public class HelloWorld{
    public static void main(String [] args){
        System.out.println("Hello world!");
    }
}
```

## compile
```bash
javac HelloWorld.java
## will create: HelloWorld.class file
```

## run
```bash
java HelloWorld
```
